STRATEGIA 3_AGENT
    Guess delle celle [x,y] tali per cui boats[x] + boats[y] == MAX, fino ad arrivare a 20

STRATEGIA 3_AGENT_1

    1) CELLE CONOSCIUTE BOAT

    IF top/bottom/left/right
        IF nave_da_4 affondata AND navi_da_3 affondate                                          AND not(navi_da_2 affondate)
            Guess su cella -1
            incremento contatore navi_da_2 affondate
        ELSE IF nave_da_4 affondata AND navi_da_2 affondate                                     AND not(navi_da_3 affondate)
            Guess su cella -1
            Guess su cella -2
            incremento contatore navi_da_3 affondate
        ELSE IF nave_da_4 affondata                                                             AND not(navi_da_2 affondate) AND not(navi_da_3 affondate)
            Guess su cella -1
            Fire su cella -2
            IF cella colpita water
                incremento contatore navi_da_2 affondate
            ELSE
                incremento contatore navi_da_3 affondate
        ELSE IF navi_da_3 affondate AND navi_da_2 affondate                                     AND not(navi_da_4 affondate)
            Guess su cella -1
            Guess su cella -2
            Guess su cella -3
            incremento contatore nave_da_4 affondata
        ELSE IF navi_da_3 affondate                                                             AND not(navi_da_2 affondate) AND not(nave_da_4 affondate)
            Fire su cella -1
            IF cella colpita last
                incremento contatore navi_da_2 affondate
            ELSE
                Guess su cella -2
                Guess su cella -3
                incremento contatore navi_da_4 affondate
        ELSE IF navi_da_2 affondate                                                             AND not(navi_da_3 affondate) AND not(nave_da_4 affondate)
            Guess su cella -1
            Guess su cella -2
            Fire su cella -3
            IF cella colpita last
                Guess su cella -3
                incremento contatore navi_da_4 affondate
            ELSE
                incremento contatore navi_da_3 affondate
        ELSE IF not(navi_da_2 affondate) AND not(navi_da_3 affondate) AND not(nave_da_4 affondata)
            Guess su cella -1
            Fire su cella -2
            IF cella colpita water
                incremento contatore navi_da_2 affondate
            ELSE IF cella colpita last
                incremento contatore navi_da_3 affondate
            ELSE IF cella colpita middle
                Guess su cella -3
                incremento contatore nave_da_4 affondata

    ELSE IF middle
        Fire su cella adiacente -1 con la probabilità maggiore
        IF cella colpita last
            Guess cella adiacente opposta (50% che sia una nave da 3 o da 4)


    2) CALCOLO CELLA PIU' PROBABILE NON ANCORA VISTA CHE SIA LONTANA DI ALMENO UNA CASELLA DA CELLE GIA' VISTE (fino ad esaurimento fire)

    Fire su cella scelta
    IF cella colpita != water
    Torna a casistica cella conosciuta


STRATEGIA 3_AGENT_2

    1) REASONING AND INFERENCE (CERTAINTY)

        1.1) CICLO DI CONTROLLO

                PER OGNI K-CELL/GUESS
                    k-per-row--
                    k-per-col--
                
                PER OGNI fila di 2 boat (k-cell o guess)
                    IF (circondate da [water OR not exist])
                        boat_2--

                Se esiste una fila di 4 boat (k-cell o guess)
                    boat_4--

                PER OGNI fila di 3 boat (k-cell o guess)
                    IF circondate da [water OR not exist]
                        boat_3--

                IF boat_4 == 0
                    PER OGNI fila di 3 boat (k-cell o guess)
                        boat_3--
                
                IF boat_3 == 0
                    Se esiste fila da 3 non ancora vista, è quella da 4
                        IF k-cell last
                            guess cella -3 da last
                        (ELSE
                            Guess cella laterale più probabile) // BAH FORSE
                
                IF boat_2 == 0
                    PER OGNI fila da 2 con K-cell last o da cui è possibile inferire la direzione
                        guess cella -2
                
                IF boat_1 == 0
                    ?


                PER OGNI k-per-row E PER OGNI k-per-col == 0
                    assert k-cell water per ogni cella non conosciuta
                    conta le celle conosciute e ricalcola i vari boat_n
                
                PER OGNI ((k-per-row != 0 AND k-per-row == count_celle_disponibili di quella riga) 
                    OR
                    (k-per-col != 0 AND k-per-col == count_celle_disponibili di quella colonna))
                    
                    guess delle celle disponibili
                    boat_n--

        1.2)
        
            SWITCH (k-cell){

            Generali:

                IF due last opposti sono distanti 0
                    boat_2--

                IF due last opposti sono distanti 1
                    boat_3--
                
                IF due last opposti sono distanti 2
                    boat_4--
                
                IF last e middle sono distanti 1
                    guess cella -1 da middle
                    boat_4--

                IF due middle sono distanti 0
                    guess laterali
                    boat_4--

            case last:

                guess cella -1
                IF c'è un middle a fianco a distanza 0
                    guess cella -2

                IF boat_4 == 0 AND boat_2 == 0 AND boat_3 != 0
                    guess cella -2
                    boat_3--

                IF boat_3 == 0 AND boat_2 == 0 AND boat_4 != 0
                    guess cella -2
                    guess cella -3
                    boat_4--

            case middle:

                % INDIVIDUAZIONE DELLA DIREZIONE
                IF row == 0 OR row == 9
                    assert water per le 5 celle -1 verticali a fianco di middle
                    guess celle adiacenti orizzontali

                ELSE IF col == 0 OR col == 9
                    assert water per le 5 celle -1 orizzontali a fianco di middle
                    guess celle adiacenti verticali

                IF cell -1 o +1 in una delle due direzioni è water
                    guess celle adiacenti opposte
                    assert water per le 6 celle attorno al middle
            
            
            case sub:
                assert water per le 8 celle intorno (controllare se esistono tutte 8)
                boat_1--
        }
        


    2) KNOWLEDGE EXPANSION DA K-CELL (UNCERTAINTY)

    IF last
        Fire su cella -2

    ELSE IF middle        
        Guess celle laterali più probabili t.c. (k-per-row != 0 AND k-per-col != 0 AND not k-cell (water))
        assert k-cell water per le 10 celle laterali a quelle "conosciute"

    
    3) UNKNOWN

        IF fires > 0
            Fire su cella più probabile t.c. che non sia già stata vista e che sia ammissibile (ossia non sia stata marchiata come water)
        ELSE
            Guess di tutte le celle rimanenti fino ad esaurimento, da quella più probabile a quella meno
            Non modificare i valori di k-per-row e k-per-col
